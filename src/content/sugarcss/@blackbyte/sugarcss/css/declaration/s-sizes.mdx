# s-sizes

<div class="docmap_type"><span class="docmap_type-label">Type:</span><span class="docmap_type-value">Declaration</span></div>

<div class="docmap_namespace">@blackbyte.sugarcss.css.declaration</div>
<div class="docmap_description typo-lead typo-format">
This variable allows you to declare the sizes values to use in your css.
You can either declare a min and max size value and an easing function to be used when you
ask for a size value with a number.
</div>

<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {
     --s-sizes: 0 80px;
}

.my-element {
   padding: s-size(10); // 80px / 100
10 = 8px
   padding: s-size(100); // 80px / 100
100 = 80px
}
```
</s-code>
