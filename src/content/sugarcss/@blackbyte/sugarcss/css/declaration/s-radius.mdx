<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {
   --s-radius-default: 10px;
   --s-radius-special: 20px 10px 12px 34px;
}

.my-element {
   border-radius: s-radius();
   border-radius: s-radius(special);

  // or using a mixin
  @s-radius(special);
}
```
</s-code>
