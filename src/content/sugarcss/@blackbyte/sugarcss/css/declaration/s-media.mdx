<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {
   --s-media-mobile: 0 768px;
   --s-media-tablet: 769px 1024px;
   --s-media-desktop: 1025px;
   --s-media-wide: 1440px;
}

.my-element {
   color: s-color(accent);

   @media mobile {
       color: red;
   }

   @media lt-tablet {
       color: blue;
   }
}
```
</s-code>
