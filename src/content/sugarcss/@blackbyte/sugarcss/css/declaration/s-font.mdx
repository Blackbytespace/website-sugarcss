<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {
   --s-font-default: s-font-family(sans);
   --s-font-lead: s-font-family(sans);
   --s-font-title: VT323 600;
   --s-font-code: s-font-family(code) 300;
}

.my-element {
   font: s-font(default);
   font: s-font(lead);
}
```
</s-code>
