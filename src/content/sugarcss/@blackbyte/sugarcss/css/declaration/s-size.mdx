<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {

     --s-size-small: 10px;
     --s-size-medium: 20px;
     --s-size-large: 40px;
}

.my-element {
   padding: s-size(small); // 10px
   padding: s-size(medium); // 20px
   padding: s-size(large); // 40px
}
```
</s-code>
