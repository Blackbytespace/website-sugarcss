# s-grid

<div class="docmap_type"><span class="docmap_type-label">Type:</span><span class="docmap_type-value">AtRule</span></div>

<div class="docmap_namespace">@blackbyte.sugarcss.css.rule</div>
<div class="docmap_description typo-lead typo-format">
This rule allows you to apply a grid layout easily.
You can either apply a registered grid, or directly
pass the grid layout you want to apply.
Here&#39;s the &quot;special&quot; characters you can use in your grid layout:
- &quot;_&quot; mean a new line in the grid layout.
- &quot;.&quot; mean an empty cell in the grid layout.
</div>

<div class="docmap_examples">
## Example
<s-code language="css" header>
```css
:root {
   --s-grid-default: 1 1 2 _ 3 3 3;
   --s-grid-2cols: 1 2;
}

.my-element {
   @s-grid(2cols);

  // or with an inline layout
  @s-grid('1 1 2 _ 3 3 3', 20px);
}
```
</s-code>
