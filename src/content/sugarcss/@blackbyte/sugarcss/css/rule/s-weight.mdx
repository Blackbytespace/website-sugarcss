# s-weight

<div class="docmap_description typo-lead typo-format">
This rule allows you to get a font weight from the registered ones or from the default ones that are:

- thin: 100
- extra-light: 200
- lighter: 200
- light: 300
- regular/normal: 400
- medium: 500
- semi-bold: 600
- bold: 700
- bolder: 800
- extra-bold: 800
- black: 900
</div>

## Example
<s-code language="css" header>
```css
:root {
   --s-weight-boldeeeeeerrrrrr: 9000;
}

.my-element {
   @s-weight(bold); // font-weight: 700;
   @s-weight(light); // font-weight: 300,
   @s-weight(boldeeeeeerrrrrr); // font-weight: 9000;
}
```
