# s-font

<div class="docmap_description typo-lead typo-format">
This rule allows you to apply a registered font easily with all his properties like:
- `family`: The font family like &quot;sans-serif&quot;, &quot;serif&quot;, etc...
- `weight`: The font weight like &quot;normal&quot;, &quot;bold&quot;, etc...
- `style`: The font style like &quot;normal&quot;, &quot;italic&quot;, etc...
- `variant`: The font variant like &quot;normal&quot;, &quot;small-caps&quot;, etc...
- `letter-spacing`: The font letter spacing like &quot;normal&quot;, &quot;0.1em&quot;, etc...
</div>
