# s-fit
<div class="docmap_metas">
<div class="docmap_type"><span class="docmap_type-label">Type:</span><span class="docmap_type-value">AtRule</span></div>
<div class="docmap_status"><span class="docmap_status-label">Status:</span><span class="docmap_status-value -stable">stable</span></div>

<div class="docmap_namespace">@blackbyte.sugarcss.css.rule</div>
<div class="docmap_description typo-lead typo-format">
This rule allows you to make the element fit the parent element easily.
You can choose the `position` you want as well as if you want to `center` the element
using `left: 50%`, `top: 50%` and `transform: translate(-50%, -50%)`.
</div>

</li>
