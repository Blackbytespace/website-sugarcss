<div class="docmap_description typo-lead typo-format">
Apply a color from the colors registered ones.
Allow to apply modifiers or shades to the color like darken, lighten, saturate, desaturate, spin, etc...
Here&#39;s the list of available modifiers:

- `lightness`: 0-100
- `lighten`: 0-100
- `darken`: 0-100
- `saturation`: 0-100
- `saturate`: 0-100
- `desaturate`: 0-100
- `hue`: 0-360
- `spin`: 0-360
- `alpha`: 0-1

The shades are registered like so: --s-shade-placeholder: --darken 10;
</div>
