## Example
<s-code language="css" header>
```css
:root {
   --s-ease-default: cubic-bezier(0.745, 0, 0.18, 1);
   --s-ease-in-out: cubic-bezier(0.86, 0, 0.07, 1);
}

.my-element {
   transition: all 1s s-ease();
   transition: all 1s s-ease(in-out);
}
```
