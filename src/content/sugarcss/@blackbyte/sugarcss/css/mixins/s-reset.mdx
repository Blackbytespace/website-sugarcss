<div class="docmap_description typo-lead typo-format">
This mixin allows you to apply the destyle css with some additional
reset rules like:
- Reset `mark` background color
- Font smoothing to &quot;antialiased&quot; for --webkit browsers
- Apply `user-select: none` to all elements except `input`, `textarea`, and `[contenteditable]`
- Remove the default outline
