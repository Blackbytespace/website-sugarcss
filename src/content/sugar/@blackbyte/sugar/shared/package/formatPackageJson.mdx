# formatPackageJson

<div class="docmap_type"><span class="docmap_type-label">Type:</span><span class="docmap_type-value">Function</span></div>

<div class="docmap_description typo-lead typo-format">
This function simply take a package.json JSON formatted object and standardize
some fields like the &quot;author&quot; one to be sure it is an object at the end,
the &quot;contributors&quot; array the same way as the &quot;author&quot; one, etc... Here&#39;s the list
of processed actions:
1. Transform the string version of the &quot;author&quot; field into an object with the properties &quot;name&quot;, &quot;email&quot; and &quot;url&quot;
2. Transform the string version of the &quot;contributors&quot; field into an object with the properties &quot;name&quot;, &quot;email&quot; and &quot;url&quot;
</div>
