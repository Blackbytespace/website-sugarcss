<div class="docmap_examples">
## Example
<s-code language="js" header>
```js
import { querySelector } from '@blackbyte/sugar/dom';
// simple query
const elm = querySelector('.a-cool-css-selector');

// get an element that is in the viewport
const elm = querySelector('.a-cool-css-selector', {
		inViewport : true
});
```
