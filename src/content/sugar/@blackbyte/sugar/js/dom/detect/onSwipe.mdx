## Example
<s-code language="js" header>
```js
import { onSwipe } from '@blackbyte/sugar/dom'
onSwipe(myCoolElm, (swipe) => {
	// check the swipe direction
	if (swipe.left) {
		// do something...
	}
	// support : left, right, up, down
	// etc...
}, {
	threshold : 50
});
```
