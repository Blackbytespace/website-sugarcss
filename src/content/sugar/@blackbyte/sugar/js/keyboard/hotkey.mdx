# hotkey
<div class="docmap_metas">
<div class="docmap_type"><span class="docmap_type-label">Type:</span><span class="docmap_type-value">Function</span></div>

<div class="docmap_namespace">@blackbyte.sugar.js.keyboard</div>
<div class="docmap_description typo-lead typo-format">
This function allows you to register a hotkey on the passed element.
You can specify the hotkey using the following syntax:
- ctrl+a
- shift+ctrl+a
- a
This function take care of avoiding to call your callback when the active element
is an input, a textarea or an editable content with the &quot;contenteditable&quot; attribute,
unless:
- the &quot;meta&quot; or &quot;ctrl&quot; key is used in the hotkey.
- the pressed key is the &quot;escape&quot; key.
</div>

## Example

