<div class="docmap_description typo-lead typo-format">
Retreive an object value using a dotted path like &quot;myObject.myProperty.myValue&quot;.
It support glob patterns like &quot;something..id&quot; and returns you a new object containing
all values with the path that matches the passed glob pattern.
