<div class="docmap_description typo-lead typo-format">
This is the same function as the &quot;Array.map&quot; but for objects. It will iterate over all the properties
of the passed object and pass the value to your process function. It will then save the property
with your processed value

<div class="docmap_examples">
## Example
<s-code language="js" header>
```js
import { map } from '@blackbyte/sugar/object';
const myObject = {
   hello: 'world',
   cat: 'Nelson'
};
map(myObject, ({value, prop}) => {
   return prop === 'hello' ? 'universe' : value;
});
{
   hello: 'universe',
   cat: 'Nelson'
}
```
