<div class="docmap_description typo-lead typo-format">
This function allows you to get an array of common text file extensions with or without the dot
Common formats includes: zip, rar, 7z, tar, gz, &#39;targz, tgz, bz2, &#39;tarbz2, tbz2, xz, &#39;tarxz, txz, iso, dmg, cab, deb, rpm, pkg, jar, war, apk
Extended formats includes: ace, alz, arc, arj, bz, cpio, egg, lha, lz, lzh, lzma, lzo, pak, sit, sitx, &#39;tar.lz&#39;, &#39;tar.Z&#39;, tbz, tlz, tz, tZ, wim, xar, z, zipx
